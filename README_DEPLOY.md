# 📊 학생 성적 조회 시스템

회계정보시스템 수강생들을 위한 온라인 성적 조회 시스템입니다.

## 🎯 기능

- **학생 성적 조회**: 학번 + 이름으로 안전한 성적 확인
- **보안**: 정확한 학번과 이름 매칭 필요
- **관리자 기능**: 조회 기간 설정 및 관리
- **반응형 웹**: 모바일/데스크톱 모든 기기 지원

## 🔒 보안 특징

- 쿼리 로그 없음 (개인정보 보호)
- 정확한 학번 + 이름 매칭만 허용
- 관리자 설정 조회 기간 제한

## 📱 사용 방법

1. **학생**: 학번(9자리)과 이름을 정확히 입력
2. **관리자**: `/admin` 경로에서 시스템 관리

## 🛠 기술 스택

- **Backend**: Flask (Python)
- **Frontend**: HTML, CSS, JavaScript
- **배포**: Vercel
- **데이터**: JSON
